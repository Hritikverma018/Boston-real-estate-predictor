
# 🏠 Boston Real Estate Price Predictor

A Machine Learning web application that predicts Boston housing prices using neighborhood features.  
The project is built using Python, Streamlit, and Scikit-learn.

## 🚀 Features
- Interactive web interface using Streamlit
- Predict house prices based on 13 input features
- Linear Regression Machine Learning model
- Real-time prediction results

## 🧠 Technologies Used
- Python
- Streamlit
- NumPy
- Pandas
- Scikit-learn

## 📂 Project Structure

Boston-real-estate-predictor  
│  
├── app.py  
├── boston_model.pkl  
├── requirements.txt  
└── README.md  

## ⚙️ Installation

Install required libraries:

pip install streamlit numpy pandas scikit-learn

## ▶️ Run the Application

streamlit run app.py

Then open in browser:

http://localhost:8501
